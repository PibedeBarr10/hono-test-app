# Hono app

English app

Stack:
- Hono (Bun)
- SQLite
- no ORM (because of Bun dialet and SQLite)
- Lucia Auth / Hono's Basic Auth
- Valibot / Zod
- `bun:test` / `hono/testing`
- Hono's RPC
- Arrow.js

After:
- Docker / Docker Compose / Kubernetes
- Tailwind CSS (?)

VSC Extensions:
- lit-html
